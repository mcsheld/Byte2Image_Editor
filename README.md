# Byte2Image Editor

**Визуальный редактор для создания и редактирования бинарных изображений из HEX-данных в формате, который использовался в монохромных дисплеях приборных панелей DENSO на автомобилях Kia и Hyundai.**

## 📖 О проекте

Это приложение позволяет работать с изображениями, которые представлены в виде HEX-строк, используя многослойную систему. Подходит для работы с графикой для микроконтроллеров, OLED-дисплеев, DIY-проектов и любых задач, где изображение хранится в виде массива байтов.

### Основные возможности

| Функция | Описание |
|---------|----------|
| 🖼️ **Многослойность** | Создавайте, удаляйте, переименовывайте и изменяйте порядок слоёв |
| ✏️ **Рисование** | Поточечное редактирование изображения |
| 🖱️ **Перемещение** | Drag-and-drop или клавиатурные стрелки (←→↑↓) |
| 🔄 **Сдвиг пикселей** | Пакетное смещение содержимого слоя |
| 📂 **Импорт изображений** | PNG, JPEG, BMP с настройкой порога и инверсии |
| ↩️ **Undo/Redo** | Отмена и повтор действий (до 50 шагов) |
| 💾 **HEX редактор** | Подсветка заголовка, копирование/вставка |
| 🔍 **Масштабирование** | От 5x до 30x |
| 📁 **Сохранение проектов** | Формат JSON со всеми слоями |
| 📤 **Экспорт** | Сохранение в PNG, JPEG, BMP |

### Формат данных DENSO

Редактор поддерживает специфический формат данных, используемый в монохромных дисплеях приборных панелей DENSO:

- **4 байта заголовка:** X_start, Y_block_start, X_end, Y_block_end
- **Массив байтов:** Каждый бит = один пиксель (1 — чёрный, 0 — белый)
- **Организация данных:** Построчно, блоками по 8 пикселей по вертикали

## 🚀 Быстрый старт

### 📦 Установка

### Системные требования

- **Python** 3.7 или выше
- **pip** (менеджер пакетов Python)

### Быстрая установка

1. **Клонируйте репозиторий**

``` bash
git clone https://github.com/mcsheld/Byte2Image_Editor.git
cd Byte2Image_Editor
```

2. **Установите зависимости**

``` bash
pip install numpy pillow
```

3. **Запуск программы**
``` bash
python Byte2Image_Editor.py
```
